# Dragon-Code
An open-source Codex-style application powered by free AI models.

StatusTypeScript

💡 About the project
Dragon Code is an open-source application inspired by OpenAI's Codex. It allows you to code with AI assistance, specifically configured to use the free Nvidia NIM models.

⚠️ Note: Currently, the application is hardcoded to only accept Nvidia NIM API keys. If you want to use other providers (like Claude), you will need to dive into the code and modify it yourself.

💻 Tech Stack
The application is entirely written using:

TypeScript
TSX (TypeScript with React/JSX syntax)
🌐 Important Information
Language: I am a French developer, so the application's user interface is currently entirely in French. If you fork this project, you will need to translate the UI strings yourself.
